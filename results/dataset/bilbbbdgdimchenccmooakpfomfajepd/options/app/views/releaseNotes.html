Version 2.3.0 StraTistiX fork based on StravistiX 2.2.1


- NEW graphic aRPEe Zones (~TRIMP/hour) % and minute distribution!
  Nice visualization of percent and minutes spent in different zones
- heart rate extended statistics returned to total elapsed time, from moving time
  (after consideration and case analysis I decided it is better this way, sorry for inconvenience)
- MapFlippers working again after Strava's changes that made them unfunctional
- removed some caching, because it introduced more problems than gains (also fixes CSV export)
- more extended statistics view data
- fine-tuned some default zones
- Various less important additions and fixes

Version 2.2.3 StraTistiX fork based on StravistiX 2.2.1
Mostly 'Feature' Synced with Thomas Champagne's StravistiX v2.2.1
Important changes! (Some might change in future)

- statistics now computed on weighted percentiles
  (might have big impact for activities with dynamic sampling period!)
- heart rate extended statistics now computed based on moving time, not total time
  (still not quite sure, which method is better... might decide to include both in future)

- Added CSV export (for easy analysis in spreadsheet software)
- Added Ascent speed statistics (VAM) for cycling
- Improved 'Best Splits' - click on them to highlight the part of activity they represent!
- Improved elevation data accuracy while computing extended statistics.
  (Elevation data smoothed using low pass filter with gain matched to Strava's)
- Added cadence, power and altitude data in overview table
- reordered and added some new data in extended statistics
- Weather unis preferences
- Various smaller additions and fixes, sorry for bigbug in 2.0.0.1 - settings menu not working :/

Version 2.0.0.1 StraTistiX fork based on StravistiX 2.0.0
Mostly 'Feature' Synced with Thomas Champagne's StravistiX v2.x
BIG UPDATE WITH LOTS OF GREAT NEW FEATURES :)
Rebased to StravistiX v2.0 for easier updates

- improved grade profile word description
- Added year progression (activity count, distance, elevation, time) table and chart
- Added 'Best Splits' (distance, time, elevation, hr,...) to biking activities
- export of segments as Virtual Partner (cycling: button under segment compare)
- Added segment builder link to segments page
- Added biking segment time comparisons to KOM's and PR's
- Added weather at activity date/time (wind/temp/clouds/humidity)
- more analysis data (climbing time and speed, pedalling time,...)
- Searchable common settings
- Moved leftside buttons/links under Edit/Action buttons
- Moved some leftside links to menu, reordered menu a bit
- Changed HR related computations from total to moving time
- Filtering altitude for gain computations
- Various Fixes (most important was settings page not working)

* Credits for many of new features go to tomasz.terlecki / tazmanska and Gabriel Landais / glandais !
